# Equations using KaTeX

?> This requires the KaTeX plugin to be loaded in the index page

Two dollar signs turn on 'display mode' and trigger a new line, one is for inline equations.

## Example 1

```md
$$
f(x) = \int_{-\infty}^\infty\hat f(\xi)\,e^{2 \pi i \xi x}\,d\xi
$$
```
