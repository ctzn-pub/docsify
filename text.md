# Text & Writing

## Callout Alerts

```markdown
!> **Time** is money, my friend!
```

!> **Time** is money, my friend!

## General tips

General tips like:

```markdown
?> _TODO_ unit test
```

are rendered as:

?> _TODO_ unit test

## Links

```md
[Glossary](/glossary)
```

Goes to [Glossary](/glossary)

## Checklists

```md
- [ ] foo
- bar
- [x] baz
- [] bam <~ not working
  - [ ] bim
  - [ ] lim
```

- [ ] foo
- bar
- [x] baz
- [] bam <~ not working
  - [ ] bim
  - [ ] lim
